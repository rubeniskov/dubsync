Packaging Reference

This directory contains reference packaging files and documentation
for distributing dubsync-stem across multiple platforms.

These files are NOT used directly by package managers.

Authoritative distribution locations:

Homebrew (macOS):
https://github.com/rubeniskov/homebrew-dubsync-stem

Arch Linux (AUR):
https://aur.archlinux.org/packages/dubsync-stem-bin

GitHub Releases (Linux binaries):
https://github.com/rubeniskov/dubsync-stem/releases

Directory overview:

homebrew/
  Reference Homebrew formula (read-only copy)

aur/
  Reference PKGBUILD for AUR (read-only copy)

Notes:

- Actual Homebrew and AUR repositories live outside this repository.
- This directory exists to document packaging approaches and help contributors.
- No binaries or release artifacts should be committed here.
